# Hello Example

This is a small compiling Gest app that uses only public APIs.

It demonstrates:

- root and feature modules
- singleton constructor injection
- controller metadata generated by `gest generate`
- DTO binding from path, query, and JSON body
- OpenAPI JSON at `/openapi.json`
- Swagger UI at `/docs`
- in-memory tests with `gesttest`

## Run

```bash
rtk go run ./examples/hello/cmd/api
```

Then call:

```bash
curl 'http://localhost:3000/users/123?expand=true'
curl -X POST 'http://localhost:3000/users/' \
  -H 'Content-Type: application/json' \
  -d '{"name":"Grace Hopper","email":"grace@example.test"}'
```

## Generate Metadata

The generated file is checked in at `internal/users/users_gest.gen.go`.

After changing controller decorators, run:

```bash
rtk go run ./cmd/gest generate --root examples/hello
```

From inside this directory, the equivalent CLI workflow is:

```bash
go run ../../cmd/gest generate
go run ../../cmd/gest build
```

## Test

```bash
rtk go test ./examples/hello/...
```

The tests use `github.com/r6m/gest/gesttest` and do not open network ports.
